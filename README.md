# 🌦️ Weather App (Aplicación del Clima)

## 📖 Descripción
Aplicación del clima desarrollada en Python que utiliza la API de Open-Meteo para obtener datos meteorológicos en tiempo real.  

Permite consultar el clima actual de una o varias ciudades ingresadas por el usuario. La aplicación primero obtiene las coordenadas mediante la API de geocodificación y después consulta los datos del clima actual, mostrando resultados claros y estructurados.

---

## 🚀 Características

- Consulta del clima en tiempo real  
- Soporte para múltiples ciudades  
- Muestra:
  - Temperatura (°C)
  - Velocidad del viento  
- Validación de entrada (evita valores vacíos)  
- Manejo de errores (ciudades no encontradas o fallas de API)  
- Código estructurado y fácil de mantener  

---

## 🛠️ Requisitos

- Python 3.8 o superior  
- Librería `requests`  

Instalar dependencias:

```bash
pip install requests
```
